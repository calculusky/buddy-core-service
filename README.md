## Description

**Buddy Saving Service**

This service is a core service for the Buddy Savings App.

### Tech/Tools Used

-   NodeJs (Version 16.15.0)
-   Sequelize (Version 6.27.0)
-   MySQL (Version 8.0)
-   TypeScript (Version4.9.4)
-   Webpack (Version 5.75.0)

## Installation

```bash
$ pnpm install
```

## Running the app

```bash

# watch mode
$ pnpm run start:dev

# production mode
$ npm run  start:prod
```
